# Quarkus AI Integration

This is a demo that shows how to migrate a Quarkus application that exposes logic using a REST API to accept also chat-like text to do similar things.

Bob migrates the application to use LangChain4j framework to connect to LLMs using AI Services and Tools.

If you are interested you can [![Watch the recording](https://img.youtube.com/vi/nTQUwghvy5Q/default.jpg)](https://youtu.be/nTQUwghvy5Q) so you can see the whole process.

`Implementation Journey Overview` folder has all the instructions to reproduce the video, and in `Prompt Templates` folder you can find all the prompts used in the demo.
