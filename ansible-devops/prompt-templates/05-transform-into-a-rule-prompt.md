# Transform a Playbook into an Ansible Rule

**Mode:** Ansible Developer

**Context:**

The playbook is not reusable; if we want to reuse it, one option is transform it into an Ansible Rule.

**Prompt:**

```
Can you transform this playbook into an Ansible Role?
```

**Result:**

Bob creates an Ansible Rule directory layout and moves all playbook pieces to the correct locations.
Finally, modifies the playbook to invoke the created rule.

Moreover, creates a metadata file so you can share it using Ansible Galaxy.
