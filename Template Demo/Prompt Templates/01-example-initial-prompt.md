# Example Initial Prompt

**Mode:** Code

**Context:**
This is an example prompt template showing how to structure your initial request to IBM Bob. Replace this with your actual demo content.

**Prompt:**
```
I need help creating a simple React component that displays a user profile card. The component should:
- Accept props for name, email, and avatar URL
- Display the information in a clean, card-style layout
- Include basic styling with CSS
- Be responsive for mobile devices

Please create the component file and associated styles.
```

**Result:**
Bob created the React component file and CSS styling as requested. The component was functional and included responsive design considerations.

**Follow-up Actions:**
- Tested the component in a local development environment
- Requested minor styling adjustments
- Added PropTypes validation

## Tips for Writing Effective Prompts

1. **Be Specific:** Clearly state what you want to accomplish
2. **Provide Context:** Explain the project background and constraints
3. **Set Expectations:** Describe the desired outcome
4. **Choose the Right Mode:** Select Code, Plan, Ask, or Advanced based on your needs
5. **Iterate:** Don't expect perfection on the first try - refine as needed
