# Prompt Templates

This folder contains the prompts used to interact with IBM Bob during this demo.

## Purpose

Prompt templates help others:
- Understand how to effectively communicate with Bob
- Reproduce similar results in their own projects
- Learn best practices for prompt engineering with Bob

## What to Include

### Initial Prompts
- The first prompt(s) that started the demo
- Any context or background information provided
- Mode selection (Code, Plan, Ask, etc.)

### Follow-up Prompts
- Iterative refinement prompts
- Clarification questions and answers
- Course corrections or adjustments

### Custom Instructions
- Any custom instructions added to Bob's configuration
- Mode-specific settings or preferences
- Tool usage preferences

## File Naming Convention

Use descriptive names that indicate the sequence and purpose:
- `01-initial-prompt.md` - The starting prompt
- `02-refinement-prompt.md` - Follow-up refinements
- `03-final-adjustments.md` - Final tweaks

## Example Structure

```markdown
# Prompt: [Brief Description]

**Mode:** Code / Plan / Ask / Advanced / etc.

**Context:**
[Any relevant background information]

**Prompt:**
[The actual prompt text sent to Bob]

**Result:**
[Brief description of what Bob did in response]
